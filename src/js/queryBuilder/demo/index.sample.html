Angular Query Builder

CommonJs plugin for user friendly query/filter interface.

Code on Github

Overview

QueryBuilder is an Angular component to create queries and filters
  • Allows for user input/copy/paste functionality for ease query format samples.
  • Build JSON object for server restful interaction and parsing.
  • It outputs a structured JSON of rules which can be easily parsed.

Example:

Sample Queries: Copy/Paste or Write them out

Country equal `USA` AND Country equal `Canada` AND Gender equal `Male` AND Gender equal `Female` AND ( State equal `North Carolina` OR Age equal `15` )

Country equal `USA` AND Country equal `Canada` AND ((Gender equal `Male` AND Gender equal `Female`) AND (State equal `North Carolina` OR State equal `Ontario`))
